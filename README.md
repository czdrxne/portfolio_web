# PERSONAL PORTFOLIO WEBSITE

A minimalist, typographic, and performance-focused personal portfolio website built with **SvelteUI**, **JavaScript**, **TailwindCSS**, **GSAP**, and **Tabler Icons**. This website presents my work, experience, thoughts, and creative direction through a clean dark interface with smooth motion and a strong focus on typography.
